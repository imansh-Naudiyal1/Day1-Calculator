Day 3: Functions Deep Dive - Function Declarations, Expressions, and Arrow Functions

Function Types

1. Function Declaration

function add(a, b) { return a + b; }

Features: Hoisted, can be called before declaration

2. Function Expression

const multiply = function(a, b) { return a * b; };

Features: Not hoisted, assigned to variable

3. Arrow Function

const divide = (a, b) => { return b !== 0 ? a / b : 'Error'; };

Features: Concise syntax, lexical this binding
